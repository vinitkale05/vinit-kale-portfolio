# Vinit Kale - Portfolio Website 🚀

Welcome to my personal portfolio repository! This project showcases my work, skills, and passion for software development and artificial intelligence through a highly interactive and immersive 3D experience.

![Portfolio Preview](https://github.com/user-attachments/assets/3c4557e7-6392-4928-b8a9-7b2476ef4edd)

## 🌟 Overview

This portfolio is built to push the boundaries of modern web design. It features a custom 3D character that interacts with the user's scroll, providing a unique narrative journey through my career and projects.

### Key Features:
- **Immersive 3D Experience**: Integrated Three.js character model that responds to scroll progress.
- **Smooth Navigation**: Powered by Lenis for high-performance, fluid scrolling.
- **Dynamic Animations**: Custom text splitting and reveal effects built with GSAP.
- **Responsive Design**: Optimized for all devices, from desktop to mobile.
- **Project Showcase**: A horizontal-scrolling section highlighting my key contributions in AI and Full-Stack development.

## 🛠️ Tech Stack

- **Frontend**: React.js, TypeScript
- **3D Graphics**: Three.js, React Three Fiber, React Three Drei
- **Animations**: GSAP (GreenSock Animation Platform)
- **Smooth Scrolling**: Lenis
- **Styling**: Vanilla CSS with modern aesthetics
- **Build Tool**: Vite

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/vinitkale05/vinit-kale-portfolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd vinit-kale-portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

## 📂 Project Structure

```text
src/
├── components/         # React components (Landing, About, Work, etc.)
│   ├── Character/     # 3D character and Three.js logic

│   └── utils/         # Custom utilities (SplitText, GsapScroll, etc.)
├── App.tsx             # Main application entry
└── main.tsx            # React mounting point
```
